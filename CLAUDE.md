## 브랜드 정체성: ONE BLANK

ONE BLANK는 20대 여성 프리랜서를 위한 전문적이고 고급스러운 브랜드다.
모든 화면과 코드는 이 정체성에서 절대 벗어나지 않는다.

---

## 1. 시각적 무드 — Ultimate Dark Mode

| 항목 | 값 |
|---|---|
| 배경 (Primary) | `#111111` (딥 그레이) |
| 배경 (Secondary) | `#1A1A1A` |
| 배경 (Surface) | `#222222` |
| 텍스트 (Primary) | `#FFFFFF` |
| 텍스트 (Secondary) | `#AAAAAA` |
| 텍스트 (Muted) | `#555555` |
| 포인트 컬러 | `#FFFFFF` (화이트) or subtle `#333333` border |
| 금지 색상 | 화려한 채도 높은 컬러 일체 금지 |

**원칙:**
- 색상은 최소화. 계층은 밝기(opacity/brightness)로만 표현한다.
- 화려한 그라디언트, 컬러풀한 버튼, 과도한 시각 효과는 절대 사용하지 않는다.
- 여백을 충분히 활용하여 고급스러운 공간감을 만든다.

---

## 2. 철학 — 80/20 룰

**80% 차갑고 이성적인 규율**
- 명확한 정보 구조와 타이포그래피
- 군더더기 없는 레이아웃
- 기능 중심, 불필요한 장식 배제

**20% 따뜻한 공감**
- 섬세한 마이크로 인터랙션 (hover, transition 등)
- 사용자의 감정을 배려하는 UX 문구
- 차갑지 않게, 그러나 과하지 않게

---

## 3. 타겟 유저

- **연령/성별:** 20대 여성
- **직업:** 프리랜서 (디자이너, 작가, 마케터 등)
- **키워드:** 전문적, 고급스러운, 간결한, 신뢰할 수 있는
- **감성:** 과하지 않은 세련됨. 럭셔리하되 접근 가능한.

---

## 4. 타이포그래피

- 폰트는 기하학적 산세리프 계열 우선 (Pretendard, Inter, Geist 등)
- 폰트 컬러는 항상 화이트 계열, 배경과 명확한 대비 유지
- 본문: Regular / UI 강조: Medium / 헤딩: SemiBold or Bold
- 자간(letter-spacing)은 살짝 넓게. 줄간격(line-height)은 여유롭게.

---

## 5. 코딩 규칙

- 모든 UI 컴포넌트는 위 색상 팔레트를 기반으로 구현한다.
- Tailwind CSS 사용 시: `bg-[#111111]`, `text-white`, `border-[#333333]` 등으로 커스텀 값을 명시한다.
- CSS 변수 권장:
  ```css
  :root {
    --bg-primary: #111111;
    --bg-secondary: #1A1A1A;
    --bg-surface: #222222;
    --text-primary: #FFFFFF;
    --text-secondary: #AAAAAA;
    --text-muted: #555555;
    --border: #333333;
  }
  ```
- 화면 구현 시 다크 모드만 지원한다. 라이트 모드 대응은 별도 지시가 없으면 구현하지 않는다.

---

## 6. 절대 금지 사항

- 밝은 흰 배경 (`#FFFFFF`, `#F5F5F5` 등) 기본 배경으로 사용 금지
- 파란색, 빨간색, 노란색 등 채도 높은 포인트 컬러 사용 금지
- Material UI, Bootstrap 기본 테마 그대로 적용 금지
- 불필요한 애니메이션, 과도한 shadow 효과 금지
